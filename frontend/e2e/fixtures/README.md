# E2E Test Fixtures

このフォルダには、E2Eテストで使用するサンプルファイルが含まれています。

## ファイル一覧

| ファイル名 | 説明 | サイズ制限 |
|-----------|------|-----------|
| `sample-team.CHE` | チームデータのサンプルファイル | 25KB以下 |
| `sample-match.CHE` | マッチデータのサンプルファイル | 260KB以下 |

## CHEファイル形式

CHEファイルはこのアプリケーション独自のバイナリ形式です。
ゲームクライアントからエクスポートされたチーム/マッチデータを含みます。

## ファイルの取得方法

サンプルファイルはゲームクライアントから以下の手順で取得できます：

1. ゲームクライアントを起動
2. チーム/マッチデータを選択
3. エクスポート機能で `.CHE` 形式で保存

## テストでの使用方法

```typescript
import { loadSampleTeamFile, loadSampleMatchFile } from '../helpers/upload-helpers';

// チームファイルを読み込む
const teamFile = loadSampleTeamFile();

// マッチファイルを読み込む
const matchFile = loadSampleMatchFile();
```

## 注意事項

- これらのファイルは実際のゲームデータを含むため、個人情報が含まれていないことを確認してください
- ファイルサイズがアプリケーションの制限を超えないようにしてください
- 新しいサンプルファイルを追加する場合は、このREADMEも更新してください
